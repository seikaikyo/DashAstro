# 宿曜道吉日查詢擴充

## 目標

將現有的「求職離職指引」擴充為通用的「吉日查詢」功能，涵蓋傳統擇日的各種場景。

## 應用場景

| 類別 | 項目 | 宿曜道判斷依據 |
|------|------|---------------|
| **事業** | 求職面試、離職提出、開業、簽約 | 本命宿 + 七曜 + 月建 |
| **學業** | 入學、考試、補習報名 | 本命宿與文昌相關宿位 |
| **居住** | 搬家入宅、裝潢開工、購屋簽約 | 本命宿 + 方位吉凶 |
| **婚姻** | 結婚登記、婚禮、訂婚 | 雙方本命宿 + 榮親/業胎吉日 |
| **醫療** | 手術、看診、健康檢查 | 本命宿 + 避凶日 |
| **旅行** | 出國、旅遊出發 | 本命宿 + 方位 |

## 後端 API

### 新增端點

```
GET /api/sukuyodo/lucky-days/{date_str}
  ?category=career|study|housing|marriage|medical|travel
  &action=interview|resign|opening|...
```

### 回應格式

```json
{
  "success": true,
  "data": {
    "category": "career",
    "action": "interview",
    "action_name": "求職面試",
    "your_mansion": { ... },
    "lucky_days": [
      {
        "date": "2026-02-03",
        "weekday": "月曜日",
        "score": 5,
        "rating": "大吉",
        "reason": "本命宿與月曜相生，適合展現自我"
      }
    ],
    "avoid_days": [ ... ],
    "general_advice": "木性本命宿者，建議選擇上午時段..."
  }
}
```
