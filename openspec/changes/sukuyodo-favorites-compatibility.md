# 宿曜道收藏對象配對一覽

## 目標

在宿曜道頁面中，讓用戶可以直接查看自己與所有收藏對象的宿曜道相性關係，無需逐一手動輸入生日。

## 變更內容

### 條件與行為

1. **顯示條件**：
   - 用戶已設定自己的生日（`profileStore.profile.birthDate`）
   - 至少有一個關注對象有生日（`profileStore.partnersWithBirthDate.length > 0`）

2. **無資料狀態**：
   - 未設定自己生日：顯示「請先設定你的生日」提示
   - 無有生日的關注對象：顯示「請新增有生日的關注對象」提示

3. **API 呼叫**：
   - 使用 `/api/sukuyodo/compatibility` POST 計算雙人相性
   - 批次查詢所有有生日的關注對象
