# 宿曜道求職離職指引

## 需求來源

用戶反映：「有沒有合適的求職（類型）離職（日期）？可能馬上用得到」

## 功能規格

### 1. 適合職業類型

根據本命宿的元素和性格特質，推薦適合的職業類型：

| 元素 | 適合職業 |
|------|---------|
| 日 | 領導、管理、創業、表演藝術、政治 |
| 月 | 服務業、醫療護理、心理諮商、教育、藝術創作 |
| 火 | 業務銷售、運動、軍警消防、廚師、工程師 |
| 水 | 研究、寫作、金融分析、資訊科技、航海 |
| 木 | 教育、法律、環保、農業、出版媒體 |
| 金 | 金融、會計、珠寶、精密工業、品管 |
| 土 | 不動產、建築、行政管理、農業、倉儲物流 |

### 2. 求職吉日

判斷條件：
- 當日七曜與本命宿元素相生或同元素 = 吉
- 當日宿與本命宿為「榮親」「業胎」關係 = 大吉
- 避開「安壞」「危成」關係日

### 3. 離職吉日

判斷條件：
- 避開「月破」「空亡」日（簡化：避開運勢低於 50 分的日子）
- 選擇「收」「開」日（簡化：運勢高於 70 分）
- 建議選在月初或月底，避免月中

### 4. 最近 30 天的吉日列表

提供：
- 求職吉日（前 5 個）
- 離職吉日（前 3 個）
- 需避開的日期

```
GET /api/sukuyodo/career-guidance/{birth_date}
```

Response:
