# 宿曜道頁面 UI 2.0 全面重製

## 問題描述

目前宿曜道頁面存在以下嚴重問題：

1. **版面雜亂**：知識區塊（六種關係、七曜、月宿傍通曆、宿曜道）排列不一致
2. **視覺不專業**：元素間距不均、對齊不佳
3. **資訊層級混亂**：重要功能與參考資訊混在一起
4. **RWD 問題**：手機版體驗不佳

## 設計決策

經用戶確認，採用以下方案：

- **整體結構**：Tab 分頁式（運勢 / 配對 / 吉日 / 知識）
- **視覺風格**：東方密教（深褐 + 琥珀金）
- **知識區塊**：獨立 Tab + 子導航
