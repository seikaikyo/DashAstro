## Phase 2：公式說明區塊

### 本命宿計算（月宿傍通曆）

```
步驟 1：西曆 → 農曆轉換
步驟 2：根據農曆月份查「起始宿」
步驟 3：起始宿 + (日期 - 1) = 本命宿

月宿傍通曆對照表：
正月(1月) → 危宿(11)    七月 → 星宿(24)
二月 → 壁宿(13)         八月 → 角宿(0)
三月 → 婁宿(15)         九月 → 氐宿(2)
四月 → 昴宿(17)         十月 → 心宿(4)
五月 → 觜宿(19)         十一月 → 斗宿(7)
六月 → 井宿(21)         十二月 → 女宿(9)

公式：本命宿 = (起始宿 + 日 - 1) mod 27
```

### 相性計算（三九秘法）

```
步驟 1：計算兩人本命宿的「距離」
        距離 = |宿位A - 宿位B|

步驟 2：根據距離判斷關係類型
        距離 0        → 命（同宿）
        距離 9 或 18  → 業胎（前世緣）
        距離 1,3,10,12,15,17,24,26 → 栄親（最佳）
        距離 2,5,11,13,14,16,22,25 → 友衰（舒適）
        距離 4,6,21,23 → 安壊（危險）
        距離 7,8,19,20 → 危成（互補）

步驟 3：計算元素加成
        同元素 → +10 分
        相生元素 → +5 分
        （木生火、火生土、土生金、金生水、水生木）
```

- 古典文獻引用（宿曜經原文片段）

- 適合職業列表
- 相性最佳/最差的宿位

每種關係增加：
- 具體相處建議（3-5 條）
- 可能衝突點與化解方法
- 適合一起做的事
