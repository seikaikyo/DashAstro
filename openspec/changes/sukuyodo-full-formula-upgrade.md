# 宿曜道完整版公式升級

## 背景

目前宿曜道功能採用「簡化版」實作，已驗證核心計算符合原典精神。本提案規劃升級至「完整版」，提供更細緻的占術分析。

## 簡化版 vs 完整版對照

| 功能 | 簡化版（目前） | 完整版（規劃） |
|------|---------------|---------------|
| 關係類型 | 6 種（合併配對） | 11 種（獨立區分） |
| 距離類型 | 無 | 近/中/遠 三種 |
| 榮親方向 | 合併 | 區分栄/親角色 |
| 安壞方向 | 合併 | 區分安/壊角色 |
| 27宿系統 | 保留牛宿 | 可切換（含/不含牛宿） |
| 月宿驗證 | 部分驗證 | 完整驗證 + 來源標註 |

## 升級內容

**目標**：在相性分析中加入近/中/遠距離判定

**影響檔案**：
- `backend/services/sukuyodo.py`
- `backend/data/sukuyodo_mansions.json`
- `frontend/src/composables/useSukuyodo.ts`
- `frontend/src/components/sukuyodo/MatchTab.vue`

### Phase 2：方向性系統

**目標**：區分榮/親、安/壊等配對關係的方向性

**概念說明**：

```
A（角宿）→ B（亢宿）：距離 = 1
  若 1 屬於「栄」的距離集合：
    → A 對 B 是「栄」的角色（給予繁榮）
    → B 對 A 是「親」的角色（受惠親愛）
```

**實作規格**：

```python
def get_directional_relation(index_a: int, index_b: int) -> dict:
    """
    計算帶方向性的關係

    Returns:
        {
            "a_to_b": "栄",      # A 對 B 的角色
            "b_to_a": "親",      # B 對 A 的角色
            "distance_type": "near",
            "combined": "榮親"   # 合併名稱（相容舊版）
        }
    """
```

**UI 呈現**：
- 顯示「你對對方：栄」「對方對你：親」
- 提供角色對應的相處建議
- 雙人各自的成長課題

### Phase 3：27/28 宿切換系統

**目標**：支援兩種宿位系統選擇

**方案**：

| 系統 | 排除宿 | 適用情境 |
|------|--------|----------|
| 日本宿曜道（27宿-牛） | 牛宿 | 日本主流占星術 |
| 傳統系統（27宿-軫） | 軫宿 | 中華傳統系統 |
| 完整系統（28宿） | 無 | 研究/對照用途 |

**實作規格**：

```python
class SukuyodoService:
    def __init__(self, mansion_system: str = "jp_27"):
        """
        mansion_system:
            - "jp_27": 日本 27 宿（排除牛宿）
            - "cn_27": 傳統 27 宿（排除軫宿）- 目前實作
            - "full_28": 完整 28 宿
        """
        self.mansion_system = mansion_system
        self._load_mansions_for_system()
```

**UI 呈現**：
- 設定頁面提供系統選擇
- 說明不同系統的歷史背景
- 結果頁面標註使用的系統

### Phase 5：進階運勢算法

**目標**：加入更多傳統宿曜道因素

**新增因素**：

1. **九曜（Navagraha）**
   - 日、月、火、水、木、金、土 + 羅睺、計都
   - 羅睺（Rahu）：北月交點，障礙星
   - 計都（Ketu）：南月交點，解脫星

2. **十二宮（黃道十二宮）**
   - 白羊、金牛、雙子... 等
   - 與本命宿交叉分析

3. **節氣影響**
   - 二十四節氣對應運勢修正
   - 特殊節日吉凶

**實作規格**（概念）：

```python
def calculate_advanced_fortune(birth_date, target_date):
    base = calculate_daily_fortune(birth_date, target_date)  # 現有

    # 加入九曜影響
    rahu_ketu_effect = calculate_rahu_ketu(target_date)

    # 加入十二宮影響
    zodiac_effect = calculate_zodiac_influence(birth_date, target_date)

    # 加入節氣影響
    solar_term_effect = calculate_solar_term(target_date)

    return combine_effects(base, rahu_ketu_effect, zodiac_effect, solar_term_effect)
```
